# DevOps Blog

A DevOps professional documenting my journey to master Kubernetes and cloud-native technologies, sharing practical insights for beginners and advanced learners alike.

## Built with [Hydejack](https://hydejack.com)

A boutique Jekyll theme for hackers, nerds, and academics.

## Quick Start

1. Clone or download this repository
2. Run `bundle install`
3. Run `bundle exec jekyll serve`
4. Open http://localhost:4000

## Structure

- `_posts/` - Blog posts
- `_data/` - YAML data files (social.yml, strings.yml)
- `assets/img/` - Images and media
- `about.md` - About page
- `index.md` - Home page

## Writing

Write all content in Markdown. See [Hydejack documentation](https://hydejack.com/docs/) for details on styling and advanced features.
